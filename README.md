# Pharmacy Project

## Get starting 

1. clone the repo
```$xslt
$ git clone https://github.com/azizk17/pharmacySE.git
```
2. add your name to names list on App.java `src\main\java\app\App.java`
3. run the class and make sure your name is printed
4. push your change 
```$xslt
$ git add .
$ git commit -m "your comment"
$ git push
```
